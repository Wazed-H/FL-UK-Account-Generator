# FLEU-Account-Generator
Generates accounts to be used on FootLocker EU

**Requires:**
  - requests
  - BeautifulSoup (bs4)

**Instructions:**
  0. Install requirements
  1. Edit the config.json using a text editor like atom or sublime
  2. Run the script
  3. Check email and click on activation link
  4. Enjoy!
